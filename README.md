
# Крестики-нолики

Игра "Крестики-нолики" с графическим интерфейсом на Pygame.

---

## Описание

Игра для двух игроков на одном компьютере. Первый игрок играет за **X** (красные), второй за **O** (синие). Цель - собрать три своих символа в ряд по горизонтали, вертикали или диагонали.

---

## Требования

### Системные требования:
- **Python**: версия 3.7 или выше
- **Pygame**: версия 2.0 или выше
- **ОС**: Windows / Linux / macOS

```bash
python --version
# Должно показать Python 3.7.x или выше
```

## Управление

### Мышь:
| Действие | Результат |
|----------|-----------|
| Клик по клетке | Сделать ход |
| Клик "Новая игра" | Начать новую партию (счет сохраняется) |
| Клик "Сброс счета" | Обнулить счетчик и начать заново |
| Клик "Выход" | Закрыть приложение |

### Клавиатура:
| Клавиша | Действие |
|---------|----------|
| **R** | Новая игра (Reset) |
| **ESC** | Выход из приложения |

## Функционал

### Игровая логика (game.py):
- Доска 3×3 клетки
- Проверка победы (8 вариантов: 3 строки, 3 столбца, 2 диагонали)
- Проверка ничьей (все клетки заполнены)
- Автоматическая смена игроков
- Обработка ошибок (занятая клетка, неверные координаты)

### Графический интерфейс (gui.py):
- Окно 600×800 пикселей
- 3 кнопки: "Новая игра", "Сброс счета", "Выход"
- Счетчик побед: **X: 0 | O: 0 | Ничья: 0**
- Статус игры: "Ход игрока: X" или "Игрок X победил!"
- Сетка 3×3 с крупными символами (X - красные, O - синие)
- Подсказка горячих клавиш внизу экрана
- Временные сообщения об ошибках (2 секунды)
